# docgen

The purpose of `docgen.py` is to convert single sources of truth from `config/*.def`
into generated, reviewable documentation artifacts.

Current outputs:

- `docs/generated/events.md`
- `docs/generated/actors.md`
- `docs/generated/routes.md`
- `docs/generated/routes.dot`
- `docs/generated/routes.svg` when Graphviz `dot` is available
